# Database Management

This repository contains database, automation, and data workflow projects focused on SQL, Excel automation, VBA, Office Scripts, JavaScript, Power BI, and operational process improvement.

The goal is to document practical tools that automate repetitive tasks, organize information, process business data, and support administrative or analytical workflows.

## Focus Areas

- Database management
- Excel automation
- VBA development
- Office Scripts
- JavaScript and TypeScript
- Power BI
- Data processing
- Workflow automation
- Operational reporting

## Projects

### Excel Quote Generator

A VBA-based Excel automation project that generates quotation files from a base quote workbook and a policy-related data file.

Main topics:

- Excel VBA automation
- File selection and validation
- Data lookup by policy name
- Batch quote generation
- Output folder creation
- Email preparation
- Administrative workflow automation

Project folder:

```text
excel_quote_generator/
```

---

### Slip Keyword Detector

An Office Scripts project for Excel Online that scans spreadsheet data, searches for predefined keywords, highlights matches, and generates result summaries.

Main topics:

- Excel Online automation
- Office Scripts
- TypeScript
- Keyword detection
- Cell highlighting
- Results sheet generation
- Summary sheet generation

Project folder:

```text
slip_keyword_detector/
```
